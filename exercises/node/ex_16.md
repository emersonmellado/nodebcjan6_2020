# Exercise 16

* Create the following folder/file structure:
```
/ex_16
  ├── index.js
  └── index.html
  └── public
      ├── css
          └── style.css
      ├── js
          └── script.js
      ├── img
          └── foto.png
```
    
## index.html
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Static documents using Node & Express</title>
  <link rel="stylesheet" href="/css/styles.css">
  <script src="/js/script.js"></script>
</head>
<body>
  <h1>Rick & Morty!!</h1>
  <img src="/img/yeay.jpg" alt="Let's go" />
</body>
</html>
```
